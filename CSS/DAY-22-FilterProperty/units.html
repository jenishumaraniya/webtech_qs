CSS UNITS

1. px (pixel) -> Means fixed size doesn't matter screen size

2. vw -> Viewport Width

3. vh -> Viewport height

4. % -> Percentage (depends upon parent size)

5. rem -> Root Element Size (html)

1rem = 16px (by default)

This paragraph has a font size of 2rem.

6. em -> Elment Size (1em = 16px) -> It will take the reference of the parent font size.
